# Super-Mylla

## Reglur

1. Það verður max 4 X / Rauðir og max 4 O / Bláir
1. Það má færa sinn lit / merki í staðinn fyrir að setja nýtt merki / lit
1. Hver leikmaður fær 30 sekóndur og það virkar eins og skákklukka

## Hvernig á að spila

1. Byrjið á því að ýta á start takann (græni takkinn)
2. Annar hvor hlið byrjar og tíminn hjá þeim byrjar
3. Ýtu á takann sem þú vilt setja litinn þinn á
4. Ýtu á taka þar sem þú ert nú þegar með lit og á nýjan reitt til að færa litinn sinn
